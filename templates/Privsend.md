Create a clean, simple, and beautiful web application called "PrivSend" — a privacy-focused token transfer tool that allows users to send tokens anonymously across chains.

## Core Features

- Simple and elegant UI for sending tokens privately
- Support multiple chains (Base, Ethereum, Arbitrum)
- Private transfer mode using simulated ZK privacy
- Easy recipient input (address or ENS)
- Real-time transaction status
- Transaction history (private log)
- Dark modern design with clean interface
- Balance display for connected wallet
- Clear "Send Privately" and "Send Normally" options

## Requirements

- React + TypeScript + Vite
- Tailwind CSS for styling
- Modern dark theme with purple and cyan accents
- Smooth animations and loading states
- Responsive design (mobile friendly)
- Clean and minimal code structure
- Very user-friendly and intuitive UX

## Design Style
- Minimalist dark interface
- Clean cards with subtle shadows
- Professional and trustworthy look
- Fast and simple user flow (like a premium wallet send feature)

Make this app feel simple to use but premium in design. Focus on clarity, privacy, and smooth experience.
